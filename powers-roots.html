Powers and Roots

Use squares, cubes, square roots, cube roots, and simple index notation.

Big Idea

Powers are repeated multiplication; roots undo powers.

Understand it first

Square

5 squared means 5 x 5.

Cube

3 cubed means 3 x 3 x 3.

Root

The square root of 49 is the number that squares to make 49.

Method Ladder

Use this same ladder every time you meet this topic.

  1. Read the power or root symbol carefully.
  2. Write it as multiplication if needed.
  3. Calculate step by step.
  4. Use inverse checks for roots.

Worked example: Calculate 4 cubed

  1. 4 cubed means 4 x 4 x 4.
  2. 4 x 4 = 16.
  3. 16 x 4 = 64.
  4. Answer: 64.
