Adult / Teacher Check

Maths Universe does not use face scanning to guess someone’s age. A face camera check can be wrong, and it is not a safe way to prove an adult is present. Use an adult or teacher email-code approval instead.

Adult approval

Ask a parent, carer, or teacher to enter their details. The site will check the email format and create a 6-digit approval code.

No face ID used. This avoids collecting face data. A real website needs a proper email service to actually send the code.

Enter the code

Type the 6-digit code to confirm adult approval on this browser.

Why not face identification?

  • Face age guessing can be wrong.
  • Photos and cameras should not be used to prove someone is an adult without a proper privacy system.
  • GitHub Pages is a static website, so it cannot securely check identity by itself.
  • Email-code approval is safer and easier for your website right now.
