# Maths Universe - Flat GitHub Upload Version

This version has NO folders. Every file is in the main repository folder.

Upload instructions:
1. Unzip this file.
2. Open the unzipped folder.
3. Select ALL files inside it.
4. Drag them into GitHub > Add file > Upload files.
5. Commit changes.
6. Wait 1-3 minutes for GitHub Pages.
7. Open https://coolgames6x.com/mathsuniverse/
8. If it still looks old, press Ctrl+F5.
